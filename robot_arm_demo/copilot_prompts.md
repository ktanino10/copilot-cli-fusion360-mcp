# Copilot CLI プロンプト集（撮影台本付き）

動画撮影中、ターミナルにこの順番で **コピペ** するためのプロンプト集。
すべて `gh copilot` ではなく **Copilot CLI (`copilot`)** + **Fusion 360 MCP server** を前提とする。
単位は cm。

---

## Step 0: 起動（カメラに映す前）

```bash
cd ~ && copilot
```

> Copilot CLI を起動し、Fusion 360 MCP に接続済みであることをタイトルバーで確認してから録画開始。

---

## Step 1: ベース（土台）の生成 — シーン3-A

```
Fusion 360 のアクティブデザインに、ロボットアームの土台を作って。
円柱、半径 6cm、高さ 2cm、原点から +Z 方向。名前は "Base"。
```

期待される動き: `create_cylinder` + ボディ名のリネーム。

---

## Step 2: ヨー回転軸 — シーン3-B

```
土台の上に、ヨー回転軸となる円柱を載せて。
半径 2.5cm、高さ 3cm、土台の上面（z=2）から +Z に立てる。名前は "YawJoint"。
```

---

## Step 3: 上腕（リンク1） — シーン3-C

```
ヨー軸の上に、上腕にあたる直方体を作って。
3cm × 3cm × 12cm、中心は (0, 0, 11)。名前は "UpperArm"。
さらに垂直エッジに 0.3cm のフィレットをかけて。
```

---

## Step 4: 肘ピッチ軸 — シーン3-D

```
上腕の上端に、肘のピッチ回転軸となる円柱を Y 方向に通して。
半径 2.0cm、長さ 4cm、中心は (0, 0, 17)、軸は Y 方向。名前は "ElbowPivot"。
```

---

## Step 5: 前腕（リンク2） — シーン3-E

```
肘の上に、前腕にあたる直方体を作って。
2.5cm × 2.5cm × 10cm、中心は (0, 0, 22)。名前は "Forearm"。
垂直エッジに 0.3cm のフィレット。
```

---

## Step 6: グリッパー（爪） — シーン3-F

```
前腕の先端に、二本指のグリッパーを作って。
それぞれ 0.6cm × 2cm × 3cm、中心は (±1.5, 0, 28.5)。
名前は "GripperL" と "GripperR"。
```

---

## Step 7: ジョイント設定 & 動作確認 — シーン4

> 注: アセンブリ型のドキュメントが必要。事前に `ファイル > 新規デザイン` を済ませておく。

```
YawJoint と UpperArm の間に revolute joint を追加して、Z 軸まわりに回転できるようにして。
ElbowPivot と Forearm の間にも revolute joint を追加して、Y 軸まわりに回転できるようにして。
```

---

## Step 8: エクスポート — シーン5

```
今のデザインを robot_arm.f3d として
D:\workspace\fusion_copilot_demo_video\fusion_artifacts\ に書き出して。
さらに UpperArm を STL でも書き出して。
```
