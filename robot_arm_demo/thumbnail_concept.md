# サムネイル案

## レイアウト (1280 × 720)
- **左半分**: 完成したロボットアームの斜めアングル (Fusion ビューポートをスクショ)
- **右半分**: 黒背景にターミナル風プロンプトを白文字で
  ```
  > make me a
    robot arm
  ```
- **下帯**: 大文字キャッチコピー
  ```
  Fusion 360 × GitHub Copilot CLI
  ```

## カラー
- 背景: ダークグレー `#1e1e1e`
- アクセント: GitHub 緑 `#2ea44f` / Fusion オレンジ `#ff7a00`

## キャッチコピー候補
1. "Build CAD by Talking."
2. "自然言語で CAD する。"
3. "MCP × Copilot CLI = ?"

## 生成手順
1. Fusion でロボットアーム完成形を Home View → スクショ
2. Figma/Canva/GIMP でテキストとレイアウトを合成
3. 1280×720 PNG で書き出し → `thumbnail.png` として配置
